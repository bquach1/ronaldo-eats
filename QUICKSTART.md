## How AI Recommendations Work

### The Algorithm Learns From:
1. **Your Ratings** - Higher rated restaurants' cuisines are preferred
2. **Cuisine Patterns** - If you rate Italian high, you'll see more Italian
3. **Price Preferences** - Matches your typical spending level
4. **Distance** - Prioritizes closer restaurants
5. **Diversity** - Ensures variety in recommendations

## Troubleshooting

**App won't open?**
- Make sure you have Expo Go installed
- Check that your phone and computer are on the same WiFi network

**Location not working?**
- Go to Settings > Privacy > Location Services
- Enable for Expo Go
- Grant location permission when prompted

**Want to start fresh?**
- Go to Profile tab
- Scroll to bottom
- Tap "Clear All Data"
- Confirm

**Still having issues?**
- Try `npm start --clear` in terminal
- Check the DEVELOPMENT.md file for more help
